- 逆序对问题
> 给定一个数组，对于这个数组中的任意两个数，若前面一个数字大于后面一个数字，则这两个数组成一个逆序对。

例子</br>
```go
arr = [1, 3, 4, 2, 5]

1 左边比 1 大的数， 没有;
3 左边比 3 大的数， 没有；
4 左边比 4 大的数， 没有;
2 左边比 2 大的数， 3、4；
5 左边比 5 大的数， 没有；

所以逆序对个数为：sum = <3,2> + <4,2> = 2
```
