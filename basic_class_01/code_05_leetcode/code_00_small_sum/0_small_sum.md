- 小和问题
> 在一个数组中，每一个数左边比当前数小的数累加起来，叫做这个数组的小和。求一个数组的小和。

例子</br>
```go
arr = [1, 3, 4, 2, 5]

1 左边比 1 小的数， 没有;
3 左边比 3 小的数， 1；
4 左边比 4 小的数， 1、3;
2 左边比 2 小的数， 1；
5 左边比 5 小的数， 1、3、4、2；

所以小和为：sum = 1+1+3+1+1+3+4+2=16
```
